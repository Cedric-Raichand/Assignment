# Vehicle Rental Management System (VRMS)

The **Vehicle Rental Management System (VRMS)** is a simple **console-based Java application** that simulates the process of renting vehicles to customers.  
It demonstrates core **Object-Oriented Programming (OOP)** concepts such as abstraction, inheritance, interfaces, and encapsulation.
